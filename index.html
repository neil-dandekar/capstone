Abstract

Large language models are powerful but difficult to interpret and control. Our project studies neuron-level interventions in Concept Bottleneck LLMs by modifying concept-linked bottleneck activations and measuring the resulting tradeoffs between task accuracy and steerability. We provide a reproducible evaluation pipeline and a web-based interface that lets users select datasets, concepts/neuron groups, and intervention strength, then inspect metrics and qualitative before/after outputs.

Background and Motivation

Concept bottleneck approaches introduce interpretable intermediate representations (concept units) that enable post-hoc analysis and targeted intervention. We focus on using these bottleneck representations as controllable levers: we intervene on concept-linked neurons and quantify how such changes affect predictive performance and steering behavior.

Key Questions

  • Causality: Which concept-linked neurons are causally relevant to accuracy?
  • Controllability: Can we steer outputs by intervening on concept bottleneck units?
  • Tradeoff: How does steerability change as accuracy degrades under stronger interventions?

Core Functionalities

  • Dataset selection: choose a dataset or evaluation split for classification/generation.
  • Concept / neuron-group selection: intervene on interpretable concept-linked units rather than raw neuron indices.
  • Intervention controls: ablation/boosting/scaling plus strength (α) and optional top-k.
  • Evaluation: report accuracy deltas and steering metrics, plus qualitative before/after examples.
  • Visualization: clear plots for tradeoffs (accuracy vs α, steering vs accuracy) and examples.

System Architecture and Features

Design Philosophy

GitHub Pages hosts the project webpage and interactive UI. Experiments can be run offline to generate static results (plots/JSON), or connected to a separate backend later for live inference. This separation keeps the capstone website stable, reproducible, and easy to review.

User Inputs

  • Dataset: dropdown
  • Task mode: classification vs generation
  • Concept / neuron group: dropdown or multiselect
  • Intervention type: ablate / boost / scale
  • Strength (α): slider
  • Optional: top-k slider

Displayed Outputs

  • Metrics: accuracy, steering score, Δ accuracy
  • Plots: accuracy vs α, steerability vs accuracy
  • Examples: before/after generations or model responses
